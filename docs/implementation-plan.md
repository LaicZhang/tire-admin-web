## 📋 开发注意事项

### 组件复用

1. **订单管理** - 采购/销售/退货/调拨/报废/报溢/索赔单共用 `views/business/order` 组件
2. **导入导出** - 开发通用导入导出组件供各模块复用
3. **列表页面** - 基于现有表格组件开发

### 技术建议

1. **表单验证** - 使用统一的验证规则
2. **权限控制** - 利用现有权限系统控制按钮级权限
3. **API错误处理** - 统一的错误提示机制
4. **数据缓存** - 对字典、配置等数据进行缓存

### 测试要求

1. 每个模块完成后进行单元测试
2. 订单流程需要端到端测试
3. 财务相关功能需要精度测试
